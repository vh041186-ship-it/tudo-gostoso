# 🍰 Tudo Gostoso

Vitrine digital para uma loja de bolos, onde clientes podem visualizar os produtos disponíveis e realizar pedidos diretamente via WhatsApp.

## 🚀 Tecnologias

* HTML
* CSS
* JavaScript

## 📌 Funcionalidades

* Exibição de bolos disponíveis
* Layout responsivo (mobile e desktop)
* Botão de contato direto via WhatsApp
* Navegação simples e intuitiva

## 📲 Integração com WhatsApp

Os botões de pedido redirecionam automaticamente o cliente para o WhatsApp com uma mensagem pré-definida, facilitando o processo de compra.

## 🌐 Deploy
https://tudo-gostoso.vercel.app/

## 🔧 Como rodar o projeto

```bash
git clone https://github.com/vh041186-ship-it/tudo-gostoso.git
cd tudo-gostoso
```

Depois, abra o arquivo `index.html` no navegador.

## 🎯 Objetivo do projeto

Este projeto simula uma aplicação real para pequenos negócios, funcionando como uma vitrine online com foco em conversão de clientes através do WhatsApp.

## 💡 Possíveis melhorias

* Filtro de produtos (por tipo de bolo)
* Sistema de busca
* Integração com catálogo dinâmico
* Melhorias na experiência do usuário (UX)
